SPA Q&A

1) Yes, my website is a Single Page Application. All of the content is loaded with a single page load, and is ready to be used by the user. There is no page navigation, only scrolling up and down on the same page, only giving an user experience as if going through different pages.

2) SPA is new architecture, which allows us to retrieve all of the page code at once (html, javascript and css). Also, all of the appropriate resources are dynamically loaded and added to the page when needed. It allows the page app to be agile and responsive without having need to load the page over and over again.

3) Technical approaches: A lot of open source JavaScript frameworks that help with building SPAs. Amongst them: AngularJS, React, Ember, Aurelia, and more.
